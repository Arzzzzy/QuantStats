Formula

Mean:

Mean = (Sum of all values) / (Total number of values)

Median:

For an odd number of values: Median = Middle value

For an even number of values: Median = (Sum of two middle values) / 2

Mode:

There is no specific formula for calculating the mode. It is simply the value(s) that occur with the highest frequency in the data set.

Range:

Range = highest value - lowest value

Variation/standard deviation:

Standard Deviation = sqrt( (sum of (x - mean)^2) / N )

Coefficient of variation:

Coefficient of Variation = (Standard Deviation / Mean) * 100

Quartiles:

Quartiles = divides the data set into 4 parts

Interquartile Range:

Interquartile Range = upper quartile - lower quartile
